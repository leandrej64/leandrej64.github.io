Q1

Dterminer la loi de \(X_1\), puis celle de \(X_2\) sachant \([X_1 = x_1]\), puis celle de \(X_3\) sachant \([X_1 = x_1, X_2=x_2]\), etc Etant donn que la fonction de vraisemblance peut scrire \[\begin{eqnarray*} {\cal L}(\theta ; x_1, \ldots, x_n) &=& P(X_1 = x_1, \ldots X_n=x_n ; \theta) \\ &=& P(X_1 = x_1 ; \theta) \prod_{i=2}^n P(X_i = x_i | X_1 = x_1, \ldots X_{i-1}=x_{i-1} ; \theta), \end{eqnarray*}\] montrer que lestimateur de maximum de vraisemblance de \(\theta\) est toujours \(\hat{\theta}_n=X_n^*\).

\(X_1\) est toujours de loi uniforme (discrte) sur {1, , }, \({\cal U}_{\{1,\ldots,\theta\}}\), puisquaucun objet na encore t tir.

Par contre, la loi de de \(X_2\) sachant \([X_1 = x_1]\) nest plus une loi uniforme sur {1, , } mais sur \(A_2\)={1, , }/{\(x_1\)}, et \(P(X_2 = x_2 | X_1 = x_1)=\frac{nombre \, d'objets \, tirs}{nombre \, d'objets \, \, tirer}=\frac{1}{\theta-1}\) si \(x_1\) est dans {1, , } et \(x_2\) dans \(A_2\), puisquon effectue un tirage uniforme dans un ensemble de \(\theta\)-1 lments connus (car on sait que le premier objet tir est \(x_1\)).

De mme, la loi de \(X_3\) sachant \([X_1 = x_1, X_2=x_2]\) est une loi uniforme sur \(A_3\)={1, , }/{\(x_1, x_2\)} et \(P(X_3 = x_3 | X_1 = x_1, X_2 = x_2)=\frac{1}{\theta-2}\).

De manire plus gnrale, \(P(X_i = x_i | X_1 = x_1, \ldots X_{i-1}=x_{i-1} ; \theta) = \frac{1}{\theta-i+1}\) si \(\forall k, x_k\) est dans \(A_k\) et la loi de de \(X_2\) sachant \([X_1 = x_1, \ldots X_{i-1}=x_{i-1}]\) est la loi uniforme sur \(A_i\)={1, , }/{\(x_1, \ldots , x_{i-1}\)}.

Donc si \(\forall k, x_k\) est dans \(A_k\) donc max(\(x_i) \leq \theta\) : \[{\cal L}(\theta ; x_1, \ldots, x_n) = P(X_1 = x_1 ; \theta) \prod_{i=2}^n P(X_i = x_i | X_1 = x_1, \ldots X_{i-1}=x_{i-1}) = \frac{1}{\theta}*\prod_{i=2}^n \frac{1}{\theta-i+1}=\frac{1}{\theta}*\frac{1}{\theta-1}* \ldots *\frac{1}{\theta-n+1}. \] Sinon, \({\cal L}(\theta ; x_1, \ldots, x_n) = 0.\)

Le maximum de cette fonction est atteint quand \(\theta\) est minimum, donc comme max(\(x_i) \leq \theta\), il est atteint pour \(\theta = max(x_i) = X^*_n\) donc \(\hat{\theta}_n=X_n^*\).

Q2

Montrer que \(\forall k \in \{n,\ldots,\theta\}\), \(P(X_n^*=k) = {\frac{\left(_{n-1}^{k-1} \right)}{\left(_{n}^{\theta} \right)}}\). Calculer \(E[X_n^*]\) et en dduire que \(\hat{\theta}_n^{(1)}={\frac{n+1}{n}X_n^*-1}\) est un estimateur sans biais de \(\theta\).

\[P(X_n^*=k)=\frac{nombre \, de \, combinaisons \, de \, chars \, pour \, lesquelles \, X^*_n = k \, pour \, n \, chars \, observs \, parmis \, \theta \, chars \, construits}{nombre \, de \, combinaisons \, de \, chars \, possibles \, pour \, n \, chars \, observs \, parmis \, \theta \, chars \, construits}\] Pour que \(X_n^*=k\), il faut que la combinaison contienne exactement un lment gal k (ce sera le maximum), et tous les autres numros de la combinaison doivent tre strictement infrieurs k. Ainsi, il y a \(\left(_{n-1}^{k-1} \right)\) choix possibles, puisquon doit fixer un \(x_i\) gal k puis choisir les n-1 autres numros parmi les k1 numros possibles qui sont strictement infrieurs k. Comme il y a \(\left(_{n}^{\theta} \right)\) faons de rpartir n numros observs parmis \(\theta\) numros possibles, on obtient bien :

\[P(X_n^*=k)=\frac{\left(_{n-1}^{k-1} \right)}{\left(_{n}^{\theta} \right)}.\] Comme le numro maximum dune combinaison doit tre compris entre le nombre de chars (puisquil ny a pas de remise) et le nombre total de chars produit, si \(k \notin \{n,\ldots,\theta\}\), \(P(X_n^*=k) = 0.\) Donc, selon la formule ditration de Pascal (\(\sum_{k=n}^{\theta}\left(_{n}^{k} \right) = \left(_{\theta+1}^{n+1} \right)\))

\[E[X^*_n] = \sum_{k=n}^{\theta} k*P(X_n^*=k)=\sum_{k=n}^{\theta} k*\frac{\left(_{n-1}^{k-1} \right)}{\left(_{n}^{\theta} \right)}=\sum_{k=n}^{\theta} k*\frac{(k-1)!}{(n-1)!(k-n)!}*\frac{n!(\theta-n)!}{\theta!}=n*\frac{n!(\theta-n)!}{\theta!}*\sum_{k=n}^{\theta}\left(_{n}^{k} \right)=n*\frac{n!(\theta-n)!}{\theta!}*\left(_{\theta+1}^{n+1} \right)=\frac{n}{n+1}*(\theta+1).\] Donc \(X^*_n\) est un estimateur biais de \(\theta\). Et \(E[\hat{\theta}_n^{(1)}]=\frac{n+1}{n}E[X_n^*] - 1 = \theta.\) Donc \(\hat{\theta}_n^{(1)}\) est un estimateur sans biais de \(\theta\).

BiaisChars <- function(m, n, theta){
  
  # Gnration des chantillons correctement
  Echantillons <- replicate(m, sample(1:theta, n, replace = FALSE))
  
  # Calcul des estimateurs
  Xn <- apply(Echantillons, 2, max)  # Maximum de chaque chantillon
  X1 <- apply(Echantillons, 2, min)  # Minimum de chaque chantillon
 
  EMV <- Xn
  EMVSB <- (n + 1) / n * Xn - 1
  ESYM <- Xn + X1 - 1
  Eremise <- (Xn^(n + 1) - (Xn - 1)^(n + 1)) / (Xn^n - (Xn - 1)^n)
  
  # Moyennes des estimateurs
  moy_EMV <- mean(EMV)
  moy_EMVSB <- mean(EMVSB)
  moy_ESYM <- mean(ESYM)
  moy_Eremise <- mean(Eremise)
  
  # Calcul du biais empirique
  biais_theta0 <- mean(EMV - theta)
  biais_theta1 <- mean(EMVSB - theta)
  biais_theta2 <- mean(ESYM - theta)
  biais_Eremise <- mean(Eremise - theta)
  
  # Calcul des variances des estimateurs
  var_theta0 <- var(EMV)
  var_theta1 <- var(EMVSB)
  var_theta2 <- var(ESYM)
  var_theta_prime <- var(Eremise)
  
  cat("Variance de EMV:", var_theta0, "\n")
  cat("Variance de EMVSB:", var_theta1, "\n")
  cat("Variance de ESYM:", var_theta2, "\n")
  cat("Variance de Eremise':", var_theta_prime, "\n")
  
  # Affichage des rsultats
  cat("Moyenne des biais de EMV:", biais_theta0, "\n")
  cat("Moyenne des biais de EMVSB:", biais_theta1, "\n")
  cat("Moyenne des biais de ESYM:", biais_theta2, "\n")
  cat("Moyenne des biais de Eremise':", biais_Eremise, "\n")
  
  # Histogrammes
  par(mfrow = c(2,2))
  hist(EMV, probability = TRUE, main = "EMV", col = "lightblue", ylab="densit")
  abline(v = theta, col = "blue", lwd = 4)
  abline(v = moy_EMV, col = "red", lwd = 4)
  
  hist(EMVSB, probability = TRUE, main = "EMVSB", col = "lightgreen", ylab="densit")
  abline(v = theta, col = "blue", lwd = 4)
  abline(v = moy_EMVSB, col = "red", lwd = 4)
  
  hist(ESYM, probability = TRUE, main = "ESYM", col = "lightpink", ylab="densit")
  abline(v = theta, col = "blue", lwd = 4)
  abline(v = moy_ESYM, col = "red", lwd = 4)
  
  hist(Eremise, probability = TRUE, main = "Eremise", col = "lightyellow", ylab="densit")
  abline(v = theta, col = "blue", lwd = 4)
  abline(v = moy_Eremise, col = "red", lwd = 4)
}
